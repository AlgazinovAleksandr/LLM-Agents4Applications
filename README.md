### Project Philosophy

LLM-Agents are awesome tools that can be used in various applications, such as business, education, and daily-life situations. In particular, I often come up with ideas, how agents can make my life easier. This repository is a collection of such ideas, as well as their implementations

**Currently there are two separate applications in this directory:**

+ COSMETICS (in progress) - how we can use LLM-Agents to generate the completely new formulas for the cosmetics products (like shampoos, face cleansers, etc). The user comes up with an abstract idea (for instance, "shampoo that smells like teen spirit") and the agent generates a formula for it, as well as suggests thoughtful ideas to enhance the formula

+ POST_WRITER (in progress) - this agentic system can analyze the content, figure out what is missing and how to improve the writing style, suggest topics, and sometimes even generate the entire content for the user

Note that these are pet projects, and are being developed by myself, for myself, and are not intented to be monetized or be the best of their kind. However, I am still very open to suggestions and collaborations!
